# AxionMorph

AxionMorph is a futuristic Hyprland theme inspired by early **2000s blob morphism**, experimental interface design, and neon plasma aesthetics.  
It blends **liquid lime gradients**, **technical grid structures**, and **minimal cyber UI elements** to create a workspace that feels like a concept interface from a retro-future operating system.

The theme is designed for **Omarchy / Hyprland setups** and focuses on visual coherence across the desktop environment.

## Design Inspiration

AxiomMorph draws inspiration from:

- Early **2000s blob morphism UI**
- Experimental **motion graphics interfaces**
- **Cybernetic grid systems**
- Neon **signal-green plasma aesthetics**
- Retro-futuristic **system dashboards**

The result is a desktop environment that feels both **organic and technical** — fluid visual forms layered over structured digital geometry.

---

## Features

- Neon lime **plasma color palette**
- Grid-based **futuristic UI layout**
- Soft **glow accents**
- Clean **Hyprland window styling**
- Minimal **cyber interface aesthetics**
- Designed for **dark environments**
